# Farmacia Bonetti

Sistema web para la gestión de una farmacia, desarrollado en PHP y MySQL.

## Funcionalidades

- Inicio de sesión de usuarios.
- Gestión de medicamentos (alta, baja, modificación y consulta).
- Registro de ventas.
- Control de stock.
- Consulta de reportes.

## Requisitos

- PHP 8.0 o superior
- MySQL
- Apache (XAMPP o similar)

## Instalación

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/TU_USUARIO/FarmaciaBonetti.git
   ```

2. Copiar el proyecto en la carpeta `htdocs` de XAMPP.

3. Iniciar Apache y MySQL.

4. Crear la base de datos `farmaciabonetti`.

5. Importar el archivo SQL con las tablas.

6. Configurar los datos de conexión en `conexion.php`.

7. Abrir el navegador e ingresar a:

   ```
   http://localhost/FarmaciaBonetti
   ```
